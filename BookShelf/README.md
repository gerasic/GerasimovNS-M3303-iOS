# BookShelf (Лабораторная №1)

Простая реализация учета книг в Swift Playground с UI на SwiftUI.

## Как запустить

1. Откройте `BookShelf/BookShelf.playground` в Xcode.
2. Запустите `Contents.swift`.
3. Работайте через Live View:
- добавляйте книги через форму;
- удаляйте книги кнопкой `Удалить`;
- смотрите список в `List`;
- ищите по критерию (`title`, `author`, `genre`, `tag`, `year`).

## Что реализовано (обязательное)

- Add: добавление книги
- Delete: удаление книги по `id`
- List: вывод списка книг
- Search: поиск по нескольким критериям
- Архитектура через протокол `BookShelfProtocol` и отдельный сервис `BookShelf`
- Обработка ошибок без падений (`LibraryError` + сообщения в UI)

## Что реализовано дополнительно

- D1 (частично):
- валидация пустых `title`/`author`
- валидация года публикации (`1400...текущий год`)
- нормализация тегов (`trim + lowercase`)
- D2 (частично): `LibraryError` с понятными сообщениями
